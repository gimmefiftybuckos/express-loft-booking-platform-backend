Значения из .env.example совпадают с .env

Для запуска проекта необходимо:

1. Установить зависимости

```shell
npm i
```

2. Скопировать файл с переменными окружения:

```shell
cp .env.example .env
```

3. Запустить dev-сервер:

```shell
npm run dev
```
